Instrucciones de Uso y Formatos

CSV a JSON

1. Selecciona la dirección **CSV a JSON** en los controles.
2. Indica si la primera fila de tu texto CSV define los nombres de los atributos activando o desactivando la casilla **"CSV tiene cabecera"**.
3. Pega tus datos separados por comas y haz clic en **Convertir**.
4. Copia el array de objetos JSON resultante o descárgalo como un archivo `.json`.

JSON a CSV

1. Selecciona la dirección **JSON a CSV** en los controles.
2. Pega tu array JSON estructurado de objetos. La herramienta detectará automáticamente todas las llaves exclusivas para construir la fila de cabecera.
3. Haz clic en **Convertir**.
4. Si los valores contienen comas o comillas, la herramienta los escapará automáticamente para mantener la integridad tabular del archivo `.csv` final.
