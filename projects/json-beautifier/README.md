# JSON Beautifier - Mini Herramienta Web

Esta herramienta estática e interactiva te permite formatear, validar y minificar código JSON directamente en tu navegador de forma rápida, segura y 100% local.

## Funcionalidades principales

- **Formateo de JSON**: Embellece cadenas de código JSON desordenadas u obtenidas en una sola línea, haciéndolas fácilmente legibles.
- **Selector de indentación**: Permite escoger entre una indentación de 2 espacios, 4 espacios o tabulaciones para adaptarse a tus preferencias de codificación.
- **Validación instantánea**: Verifica si la sintaxis del JSON es válida y correcta. En caso de error, despliega alertas claras identificando el fallo.
- **Minificación de JSON**: Compacta y comprime el JSON reduciendo todos los espacios y saltos de línea innecesarios en un solo bloque lineal.
- **Copiado Rápido**: Botón interactivo que guarda directamente el resultado en tu portapapeles con confirmación visual rápida.
- **Limpieza de área**: Un botón rápido para reiniciar los campos y preparar la herramienta para una nueva consulta.
- **Privacidad Total**: Todo el procesamiento se realiza localmente en el lado del cliente (en tu navegador) mediante JavaScript nativo, lo que garantiza que tus datos no sean enviados a ningún servidor externo.

## Tecnologías utilizadas

- **HTML5**: Estructuración semántica de la página.
- **CSS3**: Estilos personalizados adaptados a un tema oscuro moderno (*Glassmorphism*).
- **JavaScript (Vanilla)**: Lógica interactiva para la validación, formateo, minificación y manipulaciones del portapapeles.
- **Bootstrap 5.3**: Framework utilizado para la maquetación responsiva, estructuración mediante rejillas (grid) y estilizado básico de controles.
