## Objetivo

Crear o actualizar el índice principal del portfolio en la raíz del repositorio.

## Instrucciones

1. Seguir las reglas de `.agents/rules/general-dev.md`.
2. Leer `projects.json`.
3. Crear o actualizar `index.html`.
4. El índice debe cargar `projects.json`.
5. Debe mostrar una tarjeta por cada proyecto.
6. Cada tarjeta debe mostrar:
   - título
   - descripción
   - categoría
   - estado
   - etiquetas
   - enlace al proyecto
7. Usar solo HTML, CSS y JavaScript vanilla.
8. No usar frameworks.
9. No usar dependencias externas.
10. Crear también:
   - `assets/css/portfolio.css`
   - `assets/js/portfolio.js`
11. Usar rutas relativas compatibles con GitHub Pages.
12. No crear todavía los mini proyectos.

## Resultado esperado

Dejar funcionando el índice raíz del portfolio.
